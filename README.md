🚆 Railway Control System
📦 Tech Stack:
🟢 MongoDB | ⚙️ Express.js | 📱 React Native | ☁️ Railway

A modern Railway Control System built with the MERN stack and deployed on Railway. This project enables efficient management of train schedules, routes, and station data through a powerful backend and a sleek mobile app interface.

🔧 Features
🛤️ Real-time train route management

🕓 Schedule tracking and editing

🧑‍💼 Admin panel for control and monitoring

📲 Mobile-first UI with React Native

🔐 Secure REST API with Express.js

🌐 Cloud-hosted with Railway & MongoDB Atlas

🚀 Getting Started
Clone the repo

git clone https://github.com/Otelbayev/full-stack-react-native-app.git
Install dependencies

Edit
cd backend && npm install  
cd ../mobile && npm install
Run the app
Start the backend:

npm run dev
Start the mobile app with Expo or your preferred emulator:

npx expo start
📦 Deployment
The backend is deployed on Railway
MongoDB is hosted on MongoDB Atlas
React Native app is optimized for Android & iOS platforms

📁 Project Structure

├── backend (Express + MongoDB)
└── mobile  (React Native app)
